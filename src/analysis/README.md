# Анализ данных 

Задача: спрогнозировать будет ли товар находиться в топ 15 % по рейтингу среди остальных товаров, на основе данных о продукте (готовой еды).


# Папка notebooks 

- clean_data.ipynb - Избавляемся от пустых значений или заменяем их. Сохраняем в data_clean.tsv
- preprocessing.ipynb - Преобразуем значения и данные в удобный для нас формат для последующей отправки данных в модели. Сохраняем в data_preprocess_2_attempt.tsv . Все ячейки с нулевым рейтингом (на момент сбора данных) сохраняем в data_zero_rating.tsv
- models.ipynb - Решаем нашу задачу c помощью МНК с различными типами регуляризаций.
- hypothesis.ipynb - Анализируем данные.
